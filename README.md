# Opportunity Baord
A simple web app that lets students browse and save opportunities like internships,and scholarships. It Uses SID4 = 6092 in the title and localStorage key.

## Features

- Add new opportunities with a title and default category (Internship)
- Save or remove opportunities between Available and Saved lists
- Filter opportunities by category (Internship, Scholarship, Undergraduate, Masters, PhD, Fellowships, Summits)
- Responsive layout: single column on mobile (≤480px), two columns side by side on larger screens (≥481px)
- Data saved in localStorage for persistence

## Technologies Used

- HTML5 (semantic and accessible)
- CSS3 (variables, Grid layout, responsive design)
- JavaScript (ES6, DOM manipulation, event delegation)
